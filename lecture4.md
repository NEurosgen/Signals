# Декодирование помехоустойчивых кодов

## 1. Декодирование по максимуму правдоподобия (МП)

Основа метода заключается в максимизации функции правдоподобия: $\max p(\overline{y}|\overline{c}_i)$ для $i = 1, \dots, M$. Для канала без памяти совместная вероятность факторизуется:

$$p(\overline{y}|\overline{x}) = \prod_{i=1}^{n} p(y_i|x_i)$$

Логарифмическая функция правдоподобия принимает вид:

$$\Lambda_m(\overline{y}) = \ln p(\overline{y}|\overline{c}_m) = \sum_{i=1}^{n} \ln p(y_i|c_{mi})$$

Вводится отношение правдоподобия для отдельного символа $L(y) = \ln \frac{p(y|1)}{p(y|0)}$ и вектор надежности $L(\overline{y}) = (L(y_1), \dots, L(y_n))$. Символы кодового слова преобразуются в биполярный формат $\xi_{mi} = 2c_{mi} - 1$, где:
* Логический $0 \rightarrow -1$
* Логическая $1 \rightarrow 1$

Метрика кодового слова вычисляется как скалярное произведение:

$$L_m(\overline{y}) = (\overline{\xi}_m, L(\overline{y}))$$

## 2. Двоичный симметричный канал (ДСК)

Для ДСК декодирование по МП математически сводится к поиску кодового слова с минимальным расстоянием Хэмминга $d(\overline{y}, \overline{c}_m)$ до принятого вектора. При вероятности ошибки $p_0$ отношение правдоподобия опирается на логарифм $\ln \frac{1-p_0}{p_0}$. Итоговое выражение метрики пропорционально величине:

$$n - 2d(\overline{y}, \overline{c}_m)$$

Поскольку длина кода $n$ фиксирована, максимизация метрики строго эквивалентна минимизации расстояния $d(\overline{y}, \overline{c}_m)$.

## 3. Решётчатое представление линейных блоковых кодов

Процесс декодирования формулируется как поиск кратчайшего пути в ориентированном графе — **решётке кода**.

**Свойства решётчатого графа:**
* Вершины разбиты на непересекающиеся множества, образующие временные ярусы.
* Нулевой (начальный) и последний ярусы содержат строго по одному узлу.
* Каждому ребру присваивается вес (метрика), а длина пути равна сумме весов составляющих его рёбер.

Сложность декодирования напрямую зависит от количества узлов в ярусах, что описывается профилем состояний $\tau = (\tau_0, \dots, \tau_n)$. Для построения графа кодовые слова $\overline{c}_m$ разделяются на векторы «прошлого» (past) и «будущего» (future).
