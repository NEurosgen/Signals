### Поле Галуа $GF(2^3)$
Порождающий полином: $p(x) = 1 + x + x^3$
Элементы поля (степени $\alpha$):
* $\alpha^0 = 1$
* $\alpha^1 = x$
* $\alpha^2 = x^2$
* $\alpha^3 = 1 + x$
* $\alpha^4 = x + x^2$
* $\alpha^5 = x^2 + x^3$ (и так далее)

Проверочная матрица $H$:
$H = (1, \alpha, \alpha^2, \alpha^3, \alpha^4, \alpha^5, \alpha^6)$
В бинарном виде:
$$H = \begin{pmatrix} 1 & 0 & 0 & 1 & 0 & 1 & 1 \\ 0 & 1 & 1 & 1 & 1 & 0 \\ 0 & 0 & 1 & 1 & 1 \end{pmatrix}$$

### Обнаружение и поиск ошибок
Код $(15, 12)$, где $n = 2^m - 1 = 15$.
Матрица: $H = (1, \alpha, \alpha^2, \dots, \alpha^{14})$

Принятый полином: $r(x) = c(x) + e(x)$
Синдром вычисляется как:
$$S_i = r(\alpha^i) = \sum_{l=0}^{n-1} (c_l + e_l)\alpha^{il} = \sum_{l=0}^{n-1} e_l \alpha^{il} = e(\alpha^i)$$

* **При одной ошибке** в позиции $\lambda$: $e(x) = x^\lambda$, синдром $S_1 = \alpha^\lambda$.
* **При двух ошибках** в позициях $i$ и $j$: $e(x) = x^i + x^j$.
Обозначим неизвестные позиции как $x = \alpha^i$ и $y = \alpha^j$. 
Синдромы:
$S_1 = x + y$
$S_3 = x^3 + y^3$ (в некоторых местах записей обозначено как $S_2$)

**Нахождение позиций ошибок (через формулы Виета):**
Известно, что $x^3 + y^3 = (x+y)(x^2 + y^2 + xy)$.
Разделим $S_3$ на $S_1$:
$$\frac{x^3 + y^3}{x + y} = x^2 + y^2 + xy = \frac{S_3}{S_1}$$
Поскольку в поле Галуа с характеристикой 2 выполняется $(x+y)^2 = x^2 + y^2 = S_1^2$, получаем:
$$S_1^2 + xy = \frac{S_3}{S_1} \implies xy = \frac{S_3}{S_1} - S_1^2$$

Уравнение локатора ошибок имеет вид $z^2 + \sigma_1 z + \sigma_2 = 0$, где:
* $\sigma_1 = S_1$
* $\sigma_2 = \frac{S_3}{S_1} - S_1^2$

### Пример расчета
Дано:
$S_1 = \alpha^2 + \alpha^9 = \alpha^{11}$
$S_3 = \alpha^6 + \alpha^{12} = \alpha^4$

Находим $xy$:
$$xy = \frac{S_3}{S_1} - S_1^2 = \frac{\alpha^4}{\alpha^{11}} + (\alpha^{11})^2 = \alpha^{-7} + \alpha^{22}$$
С учетом свойств поля:
$$xy = \alpha^8 + \alpha^7 = \alpha^{11}$$

Составляем уравнение локатора:
$$z^2 + \alpha^{11}z + \alpha^{11} = 0$$

Проверка корней $z = \alpha^2$ и $z = \alpha^9$:
1. Для $\alpha^2$:
   $(\alpha^2)^2 + \alpha^{11} \cdot \alpha^2 + \alpha^{11} = \alpha^4 + \alpha^{13} + \alpha^{11} = 0$
2. Для $\alpha^9$:
   $(\alpha^9)^2 + \alpha^{11} \cdot \alpha^9 + \alpha^{11} = \alpha^{18} + \alpha^{20} + \alpha^{11} = \alpha^3 + \alpha^5 + \alpha^{11} = 0$

Уравнение сходится, корни найдены верно. Ошибки произошли на позициях 2 и 9.
