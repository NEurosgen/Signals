**Лекция 10**

Рассматривается задача построения многочлена, который принимает заданные значения в нескольких точках. Пусть заданы значения функции в точках ( x_0, x_1, \dots, x_n ). Тогда можно построить интерполяционный многочлен степени не выше ( n ), который принимает эти значения.

Вводятся базисные многочлены Лагранжа:
[
L_i(x) = \prod_{j \neq i} \frac{x - x_j}{x_i - x_j}
]

Тогда интерполяционный многочлен записывается в виде:
[
P(x) = \sum_{i=0}^{n} y_i L_i(x)
]

---

Далее рассматривается конструкция многочлена через произведение:
[
G(x) = \prod_{i=0}^{n} (x - x_i)
]

Замечание:
[
G'(x_i) = \prod_{j \neq i} (x_i - x_j)
]

Отсюда можно переписать базисные многочлены:
[
L_i(x) = \frac{G(x)}{(x - x_i) G'(x_i)}
]

---

Таким образом:
[
P(x) = \sum_{i=0}^{n} y_i \cdot \frac{G(x)}{(x - x_i) G'(x_i)}
]

---

Далее вводится идея представления функции в виде:
[
F(x) = \frac{P(x)}{G(x)}
]

И рассматривается разложение на простые дроби:
[
\frac{P(x)}{G(x)} = \sum_{i=0}^{n} \frac{y_i}{(x - x_i) G'(x_i)}
]

---

**Алгоритмическая часть**

1. Строим многочлен:
   [
   G(x) = \prod (x - x_i)
   ]

2. Находим значения производной:
   [
   G'(x_i)
   ]

3. Для каждой точки считаем коэффициенты:
   [
   \frac{y_i}{G'(x_i)}
   ]

4. Собираем итоговый многочлен.

---

**Пример**

Заданы точки ( x_i ), строится:

* ( G(x) )
* вычисляются ( G'(x_i) )
* затем по формуле собирается ( P(x) )

В процессе используются таблицы значений:

* ( x_i )
* ( y_i )
* ( G'(x_i) )
* коэффициенты

---

**Ключевая идея**

Интерполяционный многочлен можно эффективно строить через:

* произведение корней ( G(x) )
* значения производной в узлах
* разложение в сумму дробей

Это позволяет избежать прямого построения базиса Лагранжа и делает вычисления более удобными.
