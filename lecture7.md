# Лекция 7. Циклические коды и алгебраические структуры

## 1. Циклические коды
Циклическому $(n, k)$-коду принадлежит приведенный порождающий полином $g(x)$ наименьшей степени.

**Теорема:** В циклических кодах полином $g(x)$ является кодовым словом наименьшей степени, и любое кодовое слово $c(x)$ делится на $g(x)$ без остатка:
$$c(x) = q(x) \cdot g(x)$$

**Следствие:** Кодовое слово делится на бином $x^n - 1$. В кольце вычетов по модулю $x^n - 1$ выполняется условие:
$$g(x)q(x) \pmod{x^n - 1} = 0$$

Проверяющий полином $h(x)$ вычисляется по формуле:
$$h(x) = \frac{x^n - 1}{g(x)}$$

Для кодового слова справедливо равенство:
$$c(x)h(x) \pmod{x^n - 1} = 0$$

## 2. Группы
Множество $G$ является группой, если на нем определена бинарная операция (обозначим ее $\times$) и выполняются следующие математические условия (аксиомы):
1. Замкнутость: для любых $a, b \in G$ выполняется $a \times b \in G$.
2. Ассоциативность: $a \times (b \times c) = (a \times b) \times c$.
3. Существование единичного элемента $e$: $a \times e = e \times a = a$.
4. Существование обратного элемента: для каждого $a \in G$ существует элемент $b \in G$ такой, что $a \times b = e$.

Число элементов в группе называется порядком группы. Если выполняется условие $a \times b = b \times a$, группа называется коммутативной (абелевой).

Множество $H \subset G$, которое само удовлетворяет аксиомам группы, является подгруппой $G$.
Смежный класс по подгруппе $H$ для элемента $g \in G$ представляет собой множество:
$$\{h \times g \mid h \in H\}$$

**Свойства смежных классов:**
* Смежные классы либо полностью совпадают, либо не пересекаются.
* Группа $G$ разбивается на совокупность непересекающихся смежных классов по заданной подгруппе $H$.
* Порядок подгруппы является делителем порядка группы.

Смежные классы по подгруппе $H$ сами образуют группу, единичным элементом которой является сама подгруппа $H$. Эта группа называется фактор-группой $G/H$.

**Пример:** Группа вычетов по модулю 6 $G = \{0, 1, 2, 3, 4, 5\}$ и подгруппа $H = \{0, 3\}$.
Фактор-группа состоит из непересекающихся смежных классов: $\{0, 3\}$, $\{1, 4\}$, $\{2, 5\}$.

## 3. Кольца и поля
**Кольцо** — множество $K$, на котором определены две бинарные операции (сложение и умножение), удовлетворяющие следующим аксиомам:
1. $K$ является абелевой группой по сложению.
2. $K$ замкнуто относительно операции умножения.
3. Выполняется ассоциативность умножения: $a(bc) = (ab)c$.
4. Выполняется дистрибутивность: $a(b+c) = ab + ac$.

Кольцо, содержащее единичный элемент по умножению, называется кольцом с единицей. Кольцо, в котором операция умножения коммутативна ($ab = ba$), называется коммутативным кольцом.

**Поле** $F$ — алгебраическая структура с операциями сложения и умножения, удовлетворяющая строгим условиям:
1. $F$ является абелевой группой по сложению.
2. Ненулевые элементы $F$ образуют абелеву группу по умножению.

Множество целых чисел по простому модулю $q$ образует поле Галуа $GF(q)$. Примеры простых полей: 
* $GF(2) = \{0, 1\}$
* $GF(5) = \{0, 1, 2, 3, 4\}$

## 4. Кольца многочленов
Элемент кольца многочленов имеет вид:
$$a(x) = a_0 + a_1x + \dots + a_mx^m$$
где коэффициенты $a_i \in GF(q)$.

Для приведенного многочлена $p(x)$ степени $m \ge 1$, множество всех многочленов степени меньше $m$ с операциями сложения и умножения по модулю $p(x)$ образует кольцо вычетов многочленов $F_{p(x)}[x]$.

**Теорема:** Кольцо $F_{p(x)}[x]$ является полем тогда и только тогда, когда $p(x)$ — простой (неразложимый) многочлен. При выполнении этого условия каждый ненулевой элемент поля имеет единственный обратный элемент по умножению.
