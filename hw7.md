# Решение задачи №1

### 1. Разложение $x^7 - 1$ над полем $F_2$

В поле характеристикой 2 выполняется равенство $-1 = 1$, следовательно, $x^7 - 1 = x^7 + 1$.  
Разложение на неприводимые множители над $F_2$:

$$x^7 + 1 = (x + 1)(x^3 + x + 1)(x^3 + x^2 + 1)$$

Обозначим базовые множители:
* $f_1(x) = x + 1$
* $f_2(x) = x^3 + x + 1$
* $f_3(x) = x^3 + x^2 + 1$

---

### 2. Формирование порождающих полиномов $g(x)$

Порождающий полином циклического кода длины $n=7$ должен являться делителем $x^7 - 1$. Всего существует $2^3 = 8$ делителей. Исключая тривиальные случаи ($g(x)=1$ и $g(x)=x^7+1$), получаем 6 нетривиальных порождающих полиномов:

1. **$g_1(x) = x + 1$**
2. **$g_2(x) = x^3 + x + 1$**
3. **$g_3(x) = x^3 + x^2 + 1$**
4. **$g_4(x) = f_1(x)f_2(x) = x^4 + x^3 + x^2 + 1$**
5. **$g_5(x) = f_1(x)f_3(x) = x^4 + x^2 + x + 1$**
6. **$g_6(x) = f_2(x)f_3(x) = x^6 + x^5 + x^4 + x^3 + x^2 + x + 1$**

---

### 3. Характеристики и классификация кодов

Параметры определяются как $[n, k, d]$, где $n=7$, $k = n - \deg g(x)$, а $d$ — минимальное кодовое расстояние.

| № | Порождающий полином $g(x)$ | $\deg g$ | Параметры $[n, k, d]$ | Тип кода |
|:--|:---------------------------|:--------:|:---------------------:|:---------|
| 1 | $x + 1$ | 1 | $[7, 6, 2]$ | Код с проверкой на чётность |
| 2 | $x^3 + x + 1$ | 3 | $[7, 4, 3]$ | Код Хэмминга |
| 3 | $x^3 + x^2 + 1$ | 3 | $[7, 4, 3]$ | Код Хэмминга |
| 4 | $x^4 + x^3 + x^2 + 1$ | 4 | $[7, 3, 4]$ | Симплекс-код (Dual Hamming) |
| 5 | $x^4 + x^2 + x + 1$ | 4 | $[7, 3, 4]$ | Симплекс-код (Dual Hamming) |
| 6 | $x^6 + x^5 + x^4 + x^3 + x^2 + x + 1$ | 6 | $[7, 1, 7]$ | Код повторения |

#### Пояснения к параметрам:
* **Код 1:** Делимость на $x+1$ означает, что сумма весов всех кодовых слов четна. $d=2$.
* **Коды 2 и 3:** Классические совершенные коды Хэмминга с $d=3$.
* **Коды 4 и 5:** Двойственные к кодам Хэмминга. Для них $d=2^{m-1} = 2^{3-1} = 4$.
* **Код 6:** Содержит только слова $\{0000000, 1111111\}$, что дает максимальное расстояние $d=7$.
