
## 1. Мультипликативная группа поля Галуа $GF(q)$
Мультипликативная группа конечного поля (поля Галуа) $GF(q)$ содержит ровно $q-1$ элементов. Она включает в себя все элементы поля, за исключением нулевого.

* **Порядок элемента:** Порядком элемента $\alpha$ называется наименьшее натуральное число $s$, при котором выполняется равенство $\alpha^s = 1$.
* **Корни уравнений:** Любой ненулевой элемент поля $GF(q)$ является корнем уравнения $x^{q-1} - 1 = 0$.
* **Малая теорема Ферма для конечных полей:** Для всех элементов поля (включая нулевой) справедливо тождество $x^q - x = 0$.
* **Порядок произведения элементов:** Если два элемента $a$ и $b$ имеют порядки $s_a$ и $s_b$ соответственно, и эти значения являются взаимно простыми числами, то порядок их произведения $a \cdot b$ в точности равен произведению их порядков: $s_{ab} = s_a \cdot s_b$.
* **Существование элементов заданного порядка:** Если число $s \cdot t$ является делителем порядка группы $q-1$, то в поле $GF(q)$ гарантированно существует элемент, порядок которого равен $s^t$.

## 2. Цикличность группы и примитивные элементы
Мультипликативная группа конечного поля Галуа всегда является циклической.

* **Примитивный (образующий) элемент:** Это такой элемент $\alpha$, последовательное возведение которого в степени от $1$ до $q-1$ генерирует все ненулевые элементы поля $GF(q)$. Порядок примитивного элемента равен $q-1$.

## 3. Характеристика поля
* **Определение:** Характеристикой конечного поля $p$ называется минимальное целое положительное число, такое что сумма $p$ единичных элементов поля равна нулю: $\underbrace{1 + 1 + \dots + 1}_{p \text{ раз}} = 0$. Для конечных полей характеристика всегда является простым числом.
* **Свойства полей характеристики $p$:** В поле с характеристикой $p$ выполняется тождество $(x + y)^p = x^p + y^p$. Это происходит потому, что все промежуточные биномиальные коэффициенты в разложении кратны $p$ и, следовательно, равны нулю по модулю $p$.

## 4. Примитивные и минимальные многочлены
* **Примитивный многочлен:** Неприводимый многочлен $p(x)$ степени $m$ над полем $GF(p)$ называется примитивным, если любой его корень в расширенном поле $GF(p^m)$ (поле полиномов по модулю $p(x)$) является примитивным элементом этого расширенного поля.
* **Минимальный многочлен:** Для заданного элемента $\alpha \in GF(p^m)$ минимальным многочленом $M(x)$ над базовым полем $GF(p)$ называется приведенный многочлен (со старшим коэффициентом, равным 1) наименьшей степени, для которого $\alpha$ является корнем, то есть $M(\alpha) = 0$.
* **Свойства минимального многочлена:**
    * $M(x)$ всегда неприводим над базовым полем $GF(p)$.
    * Если для некоторого многочлена $f(x)$ выполняется $f(\alpha) = 0$, то $f(x)$ делится на $M(x)$ без остатка.
    * Степень минимального многочлена $M(x)$ не превышает степень расширения поля $m$.
    * Если $\alpha$ — примитивный элемент поля $GF(p^m)$, то степень его минимального многочлена в точности равна $m$.

## 5. Построение расширений полей и коды максимальной длины
* **Расширение поля:** Переход от базового поля $GF(p)$ к расширенному $GF(p^m)$ осуществляется с помощью неприводимого многочлена степени $m$. Например, для построения поля $GF(2^3)$ из $GF(2)$ может использоваться примитивный многочлен $p(x) = 1 + x + x^3 = 0$. Элементы нового поля можно представлять как двоичные векторы или полиномы, а умножение выполняется по модулю $p(x)$.
* **Коды максимальной длины (М-последовательности):** Это циклические коды, которые порождаются примитивными многочленами.
    * Длина такого кода (число элементов до начала повторения последовательности) составляет $n = p^m - 1$. Для бинарного поля $GF(2)$ длина равна $n = 2^m - 1$.
    * Порождающий многочлен $g(x)$ для циклического кода вычисляется по формуле: $g(x) = \frac{x^n - 1}{h(x)}$, где $h(x)$ связан с примитивным многочленом поля $p(x)$.
